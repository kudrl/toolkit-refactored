# Time Series Analysis Tool

УСТАНОВКА: достаточно скачать директорию, распаковать локально, нажать на START_timeseriestool.bat и позволить ему запуститься. Потом откроется окно:

В правом окошке (настройки) можно выбрать, как именно анализировать файл. Программа поддерживает csv файлы; столбцы — это переменные (различные временные ряды), строки — это временные отсчеты (T1, T2, T3...). Инструмент умеет автоматически распознавать заголовки и заполнять пропуски (NaN).

Настройки:
Макс. лаг: Устанавливает глубину  окна (в количестве точек).
Порог графа (0.2): Минимальное значение метрики (по модулю), при котором связь считается значимой.
Авто-дифференцирование: Запускает тест Дики-Фуллера (ADF) для каждого ряда. Если ряд нестационарен (есть тренд), алгоритм автоматически переходит к первым разностям.
Коррекция P-value (fdr_bh): Применяет поправку Бенджамини-Хохберга при множественном тестировании гипотез.
Первичный анализ: Включает расчет базовых статистик для отчета (Hurst, энтропии, поиск пиков частот).

Сканирование window/lag/position:
Положение окна (Sliding Window): Прогоняет расчет в скользящем окне.
 Размер окна / Шаг (stride): Настраивает, сколько точек брать в одно окно и на сколько точек его сдвигать.
Скан по Lag: Строит график зависимости силы связи от размера лага. Позволяет визуально найти оптимальную задержку (например, ряд X влияет на Y через ровно 3 дня).
Куб (3D) / Матрицы для куба: Экстремально тяжелая операция. Строит 3D-поверхность: как меняется метрика при одновременном изменении размера окна, сдвига во времени и лага.
